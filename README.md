## Data Scientist: Analytics Professional Certificate (Codecademy)

This repository documents my progress through the **Data Scientist: Analytics Professional Certificate** offered by Codecademy. The program is divided into five learning components that build a comprehensive foundation in data analytics and data science.

### 1. Data Science Foundations

Learn the fundamentals of data science using **Python** and **SQL**. By the end of this section, students are able to perform exploratory data analysis (EDA) on real-world datasets.

### 2. Data Science Foundations II

Build on the fundamentals by learning **statistics**, **data visualization**, and **data cleaning** techniques. This section focuses on developing the skills needed to analyze datasets independently.

### 3. Statistics for Data Scientists

Learn statistical concepts used in data science, including **experimental design**, **hypothesis testing**, and **data visualization** to draw meaningful insights from data.

### 4. Advanced Databases

Strengthen SQL knowledge by working with advanced database concepts and writing more complex queries through hands-on practice.

### 5. BI Tools for Data Scientists

Develop business intelligence skills using **Microsoft Excel** and **Tableau** to analyze data and build interactive dashboards. The program concludes with a capstone portfolio project that demonstrates the ability to solve a real-world business problem using data.

## Certification Requirements

Each learning component includes **two assessments**, and a minimum score of **70%** is required to pass each assessment and earn the professional certificate.

## Learning Outcomes

Upon completing this program, I will have developed the skills to:

* Analyze and clean datasets using Python and SQL.
* Apply statistical methods to answer business questions.
* Create meaningful data visualizations and dashboards.
* Write advanced SQL queries for data analysis.
* Use Excel and Tableau for business intelligence reporting.
* Communicate data-driven insights to support business decision-making.
